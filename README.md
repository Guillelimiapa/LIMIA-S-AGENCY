# LIMIA'S AGENCY - Web Oficial

Página web de la agencia deportiva y representación de futbolistas **LIMIA'S AGENCY**.

## Estructura de archivos

```
├── index.html          # Página principal
├── style.css           # Estilos modernos y responsive
├── script.js           # Interacciones y animaciones
├── logo.png            # Logotipo oficial
├── images/
│   ├── representante.jpg   # Foto del representante
│   ├── jugador1.jpg        # Alex López
│   ├── jugador2.jpg        # Marco Rivas
│   ├── jugador3.jpg        # Jonas Gómez
│   ├── jugador4.jpg        # Diego Sánchez
│   ├── jugador5.jpg        # Takeshi Kimura
│   └── jugador6.jpg        # Lucas Ferreira
└── README.md
```

## Apartados

- **Inicio** - Hero con logo y presentación
- **Quién Soy** - Información de la agencia con foto del representante
- **Jugadores** - Cartera con fotos reales de los futbolistas representados
- **Método de Trabajo** - 6 pasos profesionales (diseño limpio sin iconos)
- **Contacto** - Información + mapa interactivo Leaflet en Estadio de O Couto, Ourense

## Características

- **Menú hamburguesa** en todos los dispositivos (overlay elegante)
- **Diseño responsive** optimizado para móvil
- **Fotos generadas por IA** de jugadores ficticios
- **Mapa Leaflet** con ubicación exacta del Estadio de O Couto
- **Sin formularios** (100% compatible con GitHub Pages estático)
- **Animaciones suaves** al hacer scroll

## Cómo subir a GitHub Pages

1. Crea un nuevo repositorio en GitHub
2. Sube todos los archivos y la carpeta `images/` a la rama `main`
3. Ve a **Settings > Pages** del repositorio
4. En **Source** selecciona **Deploy from a branch** → `main` → `/ (root)`
5. Guarda y espera unos minutos
6. Tu web estará en: `https://TU_USUARIO.github.io/NOMBRE_REPO/`

## Personalización

- **Reemplazar fotos**: Sustituye las imágenes de `images/` por fotos reales manteniendo los nombres de archivo
- **Actualizar datos de contacto**: Edita el email y teléfono en `index.html` dentro de la sección `#contacto`
- **Jugadores**: Modifica los nombres, posiciones y datos en el HTML y las imágenes correspondientes
