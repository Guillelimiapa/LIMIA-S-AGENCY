# LIMIA'S AGENCY - Web Oficial

Página web de la agencia deportiva y representación de futbolistas **LIMIA'S AGENCY**.

## Estructura de archivos

```
📁 tu-repo/
├── index.html              # Página principal
├── style.css               # Estilos modernos y responsive
├── script.js               # Interacciones y animaciones
├── logo.png                # Logotipo original (hero + footer)
├── logo-horizontal.svg     # Logotipo horizontal para navbar
└── 📁 images/
│   ├── representante.jpg   # Foto del representante
│   ├── jugador1.jpg        # Anselmovic Da Limia
│   ├── jugador2.jpg        # Víctor Pistarrápida JR
│   ├── jugador3.jpg        # Alvin Feghouli Melano
│   ├── jugador4.jpg        # Pipolski
│   ├── jugador5.jpg        # Freddy Papadopoulos
│   └── jugador6.jpg        # Susinho du Nascimiento
```

## Apartados

- **Inicio** - Hero con logo grande y presentación
- **Quién Soy** - Información de la agencia con foto del representante
- **Jugadores** - Cartera con fotos y datos de los futbolistas representados
- **Método de Trabajo** - 6 pasos profesionales (diseño limpio solo texto)
- **Contacto** - Email, teléfono, WhatsApp, ubicación + mapa interactivo Leaflet

## Características

- **Menú responsive**: Desplegado horizontal en ordenador, hamburguesa en móvil
- **Logo horizontal SVG** en la navbar: LIMIA'S negro + hombre traje + AGENCY verde
- **Logo original PNG** en hero y footer
- **Diseño responsive** optimizado para móvil y escritorio
- **Fotos generadas por IA** de jugadores ficticios
- **Mapa Leaflet** en Estadio de O Couto, Ourense
- **Enlaces funcionales**: mailto, tel, WhatsApp, Google Maps
- **Sin formularios** (100% compatible con GitHub Pages estático)
- **Animaciones suaves** al hacer scroll

## Cómo subir a GitHub Pages

1. Crea un nuevo repositorio en GitHub
2. Sube **todos** los archivos y la carpeta `images/` a la rama `main`
3. Ve a **Settings > Pages** del repositorio
4. En **Source** selecciona **Deploy from a branch** → `main` → `/ (root)`
5. Guarda y espera unos minutos
6. Tu web estará en: `https://TU_USUARIO.github.io/NOMBRE_REPO/`

## Personalización

- **Reemplazar fotos**: Sustituye las imágenes de `images/` por fotos reales
- **Actualizar datos de contacto**: Edita email, teléfono y ubicación en `index.html`
- **Jugadores**: Modifica los nombres, posiciones y datos en el HTML
- **Logo**: El `logo-horizontal.svg` es editable con cualquier editor de vectores (Figma, Illustrator, Inkscape)
